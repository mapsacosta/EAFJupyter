Base Images

Each experiment notebook inherits packages and configurations from a base notebook. There is a different base notebook for each OS supported by the EAF. To see which packages your notebook inherits from the base image, select your base:

  • AL9 - Alma Linux 9

  • AL8 - Alma Linux 8

  • SL7 - Scientific Linux 7
