Getting started - Choosing a Notebook

Each experiment has different notebook options when starting a server and are grouped into the following: LBNF/DUNE/ProtoDUNE, FIFE/Neutrinos, ACORN/ACCEL-AI, Cosmic Frontier, CMSLPC, and Fermi Generic SL7. In general, only the notebooks where the user has an associated experiment should be available when selecting which to open. Users will also have access to the Fermi Generic SL7 notebooks. All of the basic SL7 interactive notebooks inherit from the base image.

  • CMSLPC- CMS, LHC

  • Cosmic Frontier - LSST, MAGIS-100, DES, DarkSide, Holometer, CDMS, COUPP, Pierre Auger Observatory, Sloan Digital Sky Survey, GammeV

  • ACORN/ACCEL-AI - PIP-II, Test Beam Facility, ILC, LHC Accelerator Program, Muon Collider

  • FIFE/Neutrinos - LArIAT, NOvA, MicroBooNE, MINERvA, ICARUS, SBND, Mu2e, Muon g-2

  • LBNF/DUNE/ProtoDUNE - DUNE, LBNF

  • Fermi Generic SL7/CC8 - Everyone

Note

If you do not have access to your experiment, or have access to an experiment you are not a part of, please contact eaf-users@fnal.gov.
